# Chew's Blog

This is my blog. Thanks for visiting. 

## Dev

1. Install [ruby-devel](https://packages.fedoraproject.org/pkgs/ruby/ruby-devel/) and all other development tools.
2. Install jekyll dependencies
    ```
    bundle install
    ``` 
3. Local Serve
    ```
    bundle exec jekyll serve
    ```
